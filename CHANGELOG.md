# Changelog

## v0.4.40（2025-06-30）

当前最新正式版。

### 新增
- Ciallo 动作整帧生成，基于参考视频重建动画
- 健康提醒系统：喝水提醒、久坐提醒、休息提醒
- 提醒动效层
- 气泡避让面部

### 修复
- 角色一致性修正，锁定头身比例
- 待机锚定优化，动作切换更自然
- 勿扰模式判断逻辑完善

### 优化
- 图集从 9 行扩展至 14 行，新动作完整保留
- 安装程序重构为 UI 自绘

## v0.3.7（2025-06-21）

- 动态透明区域鼠标穿透
- 主程序自绘无边框标题栏

### 优化
- 位置保存与启动恢复
- 显示器变化后的可见区域保护
- 开关机自动启动开关
