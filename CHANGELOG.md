# Changelog

## v0.5.0（2026-07-13）

更有产品感的轻办公陪伴型桌面宠物。

### 新增
- 三步新手引导：基础互动 → 常用设置 → 喝水与久坐提醒
- 今日陪伴记录：喝水、活动、专注完成进入 AppData 用户数据
- 设置中心 UI 精修：窗口圆角、侧栏图标、按钮、滑杆、滚动条、字体层级统一

### 优化
- 自主动作从随机乱播改为按顺序低频轮换（挥手、等待、陪你看看、Ciallo）
- 调度逻辑改为"待机等待 → 完整动作 → 回待机 → 重新等待"
- 健康提醒文案从功能说明改为兔娘关心用户的语气
- Ciallo 纳入正式动作库，不再使用单独测试区

### 移除
- 用户可见的测试提醒按钮和开发调试措辞

### 数据保护
- 覆盖安装只替换程序文件，不删除设置、位置和陪伴记录
- 卸载器清理程序文件和快捷方式，保留用户数据目录

## v0.4.40（2025-06-30）

Ciallo 动作 + 健康提醒正式版。

### 新增
- Ciallo 动作整帧生成，基于参考视频重建动画
- 健康提醒系统：喝水提醒、久坐提醒、休息提醒
- 提醒动效层
- 气泡避让面部

### 修复
- 角色一致性修正，锁定头身比例
- 待机锚定优化，动作切换更自然
- 勿扰模式判断逻辑完善

### 优化
- 图集从 9 行扩展至 14 行，新动作完整保留
- 安装程序重构为 UI 自绘
