# 兔娘桌面宠物

> 一只兔娘在桌面上陪你工作、提醒你喝水休息。
> 因为你很重要，所以要照顾好自己。

---

## 关于这个项目

兔娘是一个非常火的网红博主，也是一个非常有意思的互联网主播。
很多粉丝都喜欢兔娘这个人，但不能每时每刻都看她的直播。

所以就有了这个想法：**在大家需要电脑工作的时候，让兔娘在桌面上陪伴你。**

兔娘每次下播的时候都会说"早点睡觉""注意身体"，她非常关心粉丝们的健康。
粉丝们也都知道这一点。

这个软件继承了兔娘的心意——陪你工作、提醒你喝水、久坐了起来活动一下。

> 本项目由粉丝制作，非官方出品。

## 功能特点

| 类别 | 功能 |
|---|---|
| 👋 新手引导 | 首次启动三步引导：认识基础互动 → 配置常用设置 → 设置喝水与久坐提醒 |
| 🎭 多动作动画 | 挥手、待机、Ciallo 招呼、喝水提醒、休息提醒、陪你看看等自主动作 |
| 🔔 健康陪伴 | 喝水提醒、久坐提醒、专注计时，今日陪伴记录自动保存 |
| ⚙️ 精致设置中心 | 圆角窗口、侧栏图标、一键调节大小/透明度/动画速度/开关机自启 |
| 🖥️ 透明背景 | 宠物窗口无边框透明，像真的站在桌面上 |
| 🪟 鼠标穿透 | 打开穿透模式后不干扰你操作其他软件 |
| 🌐 双语支持 | 中文（默认）/ English |
| 📌 系统托盘 | 后台驻留，不占任务栏 |
| 💾 数据保护 | 覆盖安装只替换程序文件，设置和陪伴记录不丢失 |

## 安装方法

1. 在上方「版本历史」中选择一个版本，点击「下载」
2. 下载 `.exe` 安装程序
3. 双击运行，按提示完成安装
4. 如果是从旧版本升级，覆盖安装即可，原有设置和陪伴记录不会被清除

**系统要求**：Windows 10 或 Windows 11（64 位）

## 常见问题 FAQ

**问：这是免费的吗？**
答：是的，完全免费使用。

**问：会占用很多系统资源吗？**
答：不会，程序轻量设计，正常使用占用很少的内存和 CPU。

**问：怎么退出程序？**
答：在系统托盘中找到兔娘图标，右键选择「退出」。

**问：怎么关闭开机自启？**
答：在设置中心关闭「开机自动启动」开关。

**问：覆盖安装新版本，之前的设置会丢吗？**
答：不会。V0.5 起覆盖安装只替换程序文件，你的设置、位置、陪伴记录都保留在 AppData 里。
